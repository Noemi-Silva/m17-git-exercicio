# Projeto: Análise Econômica de Municípios Brasileiros

Projeto desenvolvido como atividade prática de análise de dados e machine learning.

## Coleta de Dados
Os dados utilizados contêm informações econômicas e demográficas de municípios brasileiros.

## Modelagem
Foi utilizado o algoritmo K-Means com as variáveis PIB per capita e população estimada, após normalização dos dados.

## Conclusão
A análise permitiu identificar padrões econômicos entre os municípios, demonstrando a importância do uso de dados na tomada de decisão.
